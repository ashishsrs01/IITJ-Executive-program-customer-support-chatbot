# IITJ-Executive-program-customer-support-chatbot
# 🤖 Customer Support Assistant Chatbot

A chatbot-based support system built using **WotNot** to assist users with queries related to the **BS in Applied AI and Data Science program**.

🔗 **Live Demo:**  
https://ashishsrs01.github.io/IITJ-Executive-program-customer-support-chatbot/

---

## 📌 Overview

This project demonstrates how a chatbot can be integrated into a website to provide automated customer support.

The chatbot helps users:
- Understand program details
- Ask common questions
- Get instant responses without human support

---

## 🚀 Features

- 💬 AI-powered chatbot (WotNot)
- 🌐 Deployed on GitHub Pages
- 📱 Fully responsive design
- ⚡ Fast and lightweight interface
- 🧠 Real-time interaction with users
- 🎯 Designed as a project test/demo site

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS
- **Deployment:** GitHub Pages
- **Chatbot Platform:** WotNot
- **Integration:** Chat Widget Script

---

## 🧪 How to Use

1. Open the live website  
2. Click on the chatbot icon (bottom-right corner)  
3. Ask questions about the program  
4. Analyze responses and interaction flow  

---

## 🎯 Project Purpose

This project was created to:

- Learn chatbot integration in real-world scenarios  
- Build a portfolio-ready AI project  
- Demonstrate UI + AI tool combination  
- Simulate a real customer support system
